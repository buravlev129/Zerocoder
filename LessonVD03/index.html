Дополнительные вопросы

Ответ на вопрос 1. При этом, если уменьшить размер экрана, карточки сместятся — для экрана среднего размера вниз сместится одна карточка, для маленького экрана все карточки разместятся друг по другом.
Таким образом, Bootstrap позволяет с помощью сетки адаптировать сайт под различные размеры экрана, регулируя содержимое сайта в зависимости от размера экрана.
Также с помощью Bootstrap можно стилизовать создаваемый сайт

Ответ на вопрос 2. Домашнее задание к сегодняшнему уроку следующее: необходимо создать собственную структуру страницы из компонентов Bootstrap.
В Bootstrap есть компоненты для создания меню и основного контента. Используй их, чтобы создать базовую структуру сайта: шапку, название саайта, навигационное меню, помести на сайт основной контент (два-три блока) и футер — блок с контактной информацией о компании и ссылками на социальные сети
